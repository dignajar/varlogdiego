title: How to contribute to a Github project
tags: Github
content:

Here are the steps to contribute your code to existing Github project.

- Go to the repository via web. For example, https://github.com/`{username}`/`{project_name}`
- Click the "Fork" button at the top. This create a copy of the repository on your account.
- Now clone your new copy repository on your PC, open a terminal, and used the command `git clone`.

```
$ git clone git@github.com:{username}/{project_name}.git

# Output example
Cloning into '{project_name}'...
remote: Counting objects: 211, done.
remote: Total 211 (delta 0), reused 0 (delta 0), pack-reused 211
Receiving objects: 100% (211/211), 56.62 KiB | 0 bytes/s, done.
Resolving deltas: 100% (62/62), done.
Checking connectivity... done.
```

- Go into the new directory created

```
$ cd {project_name}
```

- Connect your local repo with the original repo.

```
$ git remote add {username} git@github.com:{username}/{project_name}.git
```

- Now you can make changes on your local repo, then make a push.

```
$ git add -A
$ git commit -m "New changes"
$ git push
```

- Now you have your local repo and github repo in sync.
- Go to your repository on Github web.
- Click on the button "New pull request".
- Click on the button "Create a pull request".
- Complete the title and the content with a description about the changes.
- Click again on the button "Create pull request".
- Now wait your friend accept this changes.
