This is a little script that used the Java client (32bits) from Juniper Network Connect. If you are running a 64bits plataform you need to install 32bits runtime libraries.

This script is based on the next links, with the difference it's very easy to install and configure.
- http://www.scc.kit.edu/scc/net/juniper-vpn/linux/
- http://mad-scientist.us/juniper.html

### Download
Github proyect: https://github.com/dignajar/dvpn

### Installation on Ubuntu
1. `$ sudo apt-get install -y wget unzip lib32z1 lib32ncurses5 lib32bz2-1.0`
2. `$ cd /tmp/`
3. `$ wget https://raw.githubusercontent.com/dignajar/dvpn/master/install_dvpn.sh`
4. `$ chmod 755 install_dvpn.sh`
5. `$ ./install_dvpn.sh {url} {realm}`

Where the {url} is the URL of your Juniper web portal and the {realm} is the group where your user is associated.
